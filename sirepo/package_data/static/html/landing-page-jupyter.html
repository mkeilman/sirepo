Community physics codes on a public HPC server —
  • Particle accelerators:
    • elegant
    • Warp
    • Synergia
    • OPAL
  • X-ray optics:
    • Synchrotron Radiation Workshop (SRW)
    • Shadow3
  • Helpful:   familiarity with JupyterHub.
  • Required:   a GitHub account
  • Required:   experience using the selected code:
    • from a Jupyter (aka IPython) notebook
    • on the command line (with no X11 features)
  • Note: the JupyterHub server requires authorization via your GitHub login. RadiaSoft does not get access to your GitHub password, nor does the server access your repositories or other personal data. The JupyterHub server uses GitHub OAuth in the minimal mode to authenticate the user name. This enables RadiaSoft to create and maintain a persistent workspace for you, easily accessed from any modern browser on any operating system.
  • Several of the codes made available on this server (SRW, Elegant, Shadow and more) can be executed from a browser-based GUI.
